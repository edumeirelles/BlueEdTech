Calculadora de Dano

A Calculadora de Dano foi uma atividade desenvolvida no curso de programação na BlueEdTech para colocar em prática o aprendizado em HTML, CSS e JavaScript. O jogo consiste em escolher um inimigo e uma arma. Cada inimigo possui um quantidade fixa de pontos de vida e cada arma conta com um número de dano fixo adicionado de um número que é gerado aleatoriamente de 1 a 10 pontos (análogo a um dado de 10 faces). O jogador deve clicar no botão "Calcular Dano" até que o dano da arma seja maior ou igual os pontos de vida do inimigo escolhido. Dependendo da arma escolhida não será possível derrotar os inimigos mais fortes. Clique aqui para acessar os códigos deste projeto no meu GitHub.

Jogo Ficção Interativa - Banda Turma2B - Projeto Final Módulo 1 na Blue Edtech

Este foi o projeto final do Módulo 1 no curso de programação na BlueEdTech. Eu e meus colegas Rafael Freitas e Jorge Teixeira, criamos um jogo de ficção interativa, onde o jogador deve escolher as ações de uma banda em turnê. O jogador deve escolher as ações na sequência correta para acessar as músicas que serão tocadas durante o show. Foram incluidos eventos aleatórios, que podem ser disparados em certos momentos, que fazem o jogador voltar ao estado inicial do jogo. Dependendo das opções escolhidas, o jogador pode receber recompensas ou punições. Ao final o jogo exibe um relatório com o desempenho do jogador. O jogo foi desenvolvido inteiramente em Python e executado no terminal do VS Code. Foi incluída a biblioteca Pygame para que fosse possível executar os arquivos de áudio. Clique aqui para acessar os códigos do projeto no meu GitHub.
